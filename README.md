# vue-number-format
Vue Number Format is used to format a number using fixed-point notation. It can be used to format a number with a specific number of digits to the right of the decimal.
## Features

- Lightweight < 2kb
- Dependency free
- Smooth migration
- Component or Directive flavour
- Format as you type
- Built-in validation
